Quiz - 14¶

Ejercicio 1¶

Desarrolle la función remover_iguales que recibe una lista numeros de enteros. La función debe devolver una lista sin los elementos repetidos y ordenados de manera ascendente.

Ejemplos:
remover_iguales([1, 2, 2, 3]) -> [1, 2, 3]
remover_iguales([1, 2, 3]) -> [1, 2, 3]
remover_iguales([1, 2, 2, 1]) -> [1, 2]

Ejercicio 2¶

Desarrolle la función encriptar que recibe una cadena frase. La función debe devolver una nueva cadena encriptada, siguiendo las siguientes reglas:

  1. Se deben retirar todas las letras repetidas de cada una de las palabras de la frase.

  2. Se deben ordenar las letras restantes de cada una de las palabras.

Ejemplo:
encriptar("anita lava la tina") -> "aint alv al aint"
Consejo: intente convertir la frase en una lista de palabras, luego intente ordenar las letras y construya una cadena con el resultado.
