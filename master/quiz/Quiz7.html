Quiz - 7¶

Ejercicio A¶

Desarrolle la función multstring que recibe como parametros una cadena s y un entero positivo n y devuelve una nueva cadena que contiene n copias de la cadena original

Ejemplo: multstring('Hola', 2) -> ‘HolaHola’

Ejercicio B¶

Desarrolle la función string_splosion que recibe una cadena s y devuelva una nueva cadena siguiendo el patron descrito en los ejemplos

Ejemplos:
string_splosion('Code') -> ‘CCoCodCode’
string_splosion('abc') -> ‘aababc’
string_splosion('ab') -> ‘aab’

Ejercicio C¶

Desarrolle la función array_count9 que recibe como parametro una lista no vacía de números enteros nums y devuelva cuántas veces aparece el número 9 en la lista

Ejemplo: array_count9([1, 99, 9]) -> 1

Ejercicio D¶

Desarrolle la función array_front9 que recibe una lista de números enteros nums y verifica si al menos uno de los primeros cuatro números es 9

Ejemplos:
array_front9([1, 2, 9, 3, 4]) -> True
array_front9([1, 2, 3, 4, 9]) -> False
array_front9([1, 2, 3, 4, 5]) -> False
array_front9([1, 2, 9]) -> True

Ejercicio E¶

Desarrolle la función hello_name que recibe como parametro un cadena representa un nombre name y devuelva un saludo con este nombre

Ejemplos:
hello_name('Bob') -> ‘¡Hola Bob!’
hello_name('Alice') -> ‘¡Hola Alice!’
hello_name('X') -> ‘Hola X!’

Ejercicio F¶

Desarrolle la función make_tags que recibe dos cadenas tag y word y devuelva una nueva cadena formateada siguiendo el patrón de los ejemplos

Ejemplos:
make_tags('i', 'Yay') -> ‘<i>Yay</i>’
make_tags('i', 'Hello') -> ‘<i>Hello</i>’
make_tags('cite', 'Yay') -> ‘<cite>Yay</cite>’

Ejercicio G¶

Desarrolle la función extra_end que recibe una cadena s de al menos dos caracteres y devuelve una nueva cadena con las dos últimas letras repetidas tres veces

Ejemplos:
extra_end('Hello') -> ‘lololo’
extra_end('abb') -> ‘bbbbbb’
extra_end('Hi') -> ‘HiHiHi’

Ejercicio H¶

Desarrolle la función new_extra_end (variante de la función del Ejercicio G) que recibe una cadena s de al menos dos caracteres y un entero positivo n y devuelva una nueva cadena con las dos últimas letras repetidas n veces

Ejemplos:
new_extra_end('Hello',3) -> ‘lololo’
new_extra_end('abb',1) -> ‘bb’
new_extra_end('Hi',5) -> ‘HiHiHiHiHi’

Ejercicio I¶

Desarrolle la función first_half que recibe una cadena s y devuelve la primera mitad de la cadena original

Ejemplos:
first_half('WooHoo') -> ‘Woo’
first_half('HelloThere') -> ‘Hello’
first_half('abcdef') -> ‘abc’

Ejercicio J¶

Desarrolle la función remove_first_last que recibe una cadena s de al menos dos caracteres y devuelva una cadena sin el primer y último carácter

Ejemplos:
remove_first_last('Hello') -> ‘ell’
remove_first_last('python') -> ‘ytho’
remove_first_last('coding') -> ‘odin’

Ejercicio K¶

Desarrolle la función left2 que recibe una cadena s de al menos dos caracteres y devuelva la cadena original rotada 2 posiciones a la izquierda

Ejemplos:
left2('Hello') -> ‘lloHe’
left2('Hi') -> ‘Hi’

Ejercicio L¶

Desarrolle la función remove_iesimo que recibe una cadena s no vacía y un entero positivo i y devuelva la cadena original sin el i-esimo carácter

Ejemplos:
remove_iesimo('Hello',1) -> ‘ello’
remove_iesimo('Hi',2) -> ‘H’
remove_iesimo('PyZombiess', 10) -> ‘PyZombies’
